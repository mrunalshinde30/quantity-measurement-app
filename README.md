# quantity-measurement-app
# UC1: Feet Measurement Equality

## Description
This use case checks whether two measurements in feet are equal.

## Logic Used
- equals() method is overridden
- Double.compare() is used for accurate comparison

## Example
Input: 1.0 ft and 1.0 ft  
Output: true

## Concepts Covered
- Object Equality
- Floating Point Comparison
- Null and Type Checking
